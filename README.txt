Descripción:

El sitio está enfocado en noticias deportivas, con secciones como “Inicio”, “Noticias”, “Contáctanos” y “Autores”. También tiene un formulario para que los usuarios nos escriban, un carrusel con imágenes, y una galería de tarjetas con noticias destacadas. Todo está hecho con Bootstrap 5.0 y adaptado para que se vea bien en celular, tablet y computadora.


Componentes usados:

Navbar (barra de navegación con menú responsive)
Cards (tarjetas para mostrar noticias)
Carousel (carrusel de imágenes con texto)
Accordion (sección de noticias adicionales que se despliegan)
Formulario (inputs, textarea y botón)
Badges (etiquetas de categoría)
Botones (btn-primary, btn-outline-primary, etc.)
Alert (mensaje informativo sobre horarios)
Footer (pie de página institucional con créditos)
Grid system (para organizar todo en columnas responsivas)
